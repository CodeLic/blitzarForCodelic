# Blitzar (Vue 3 Beta)

Generate Vue Forms and Data-tables fast with a simple JSON-like syntax ⚡️

```
npm i @blitzar/form@next
```

## Documentation

[blitzar-next.web.app](https://blitzar-next.web.app) (Vue 3 Beta)
